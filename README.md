OPPGAVE 1 INITIALISERING:

gå til mappen helpdesk ved å åpne terminalen og skriv inn følgende kommand: 
1) cd helpdesk
2) npm i / yarn install
3) npx prisma migrate / yarn prisma:migrate
4) npx prisma seed / yarn prisma:seed

5) npm run start
